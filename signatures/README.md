# Signatures

Autosubst 2 uses a variant of HOAS, customised to its expressiveness: **EHOAS**, short for **extended higher-order abstract syntax**, restricts higher-order abstract syntax to second-order and non-dependent binders. 

See Chapter 5 of **[Mechanising Syntax with Binders in Coq](https://www.ps.uni-saarland.de/Publications/details/Stark:2020:Mechanising.html)**  [*(pdf)*](https://www.ps.uni-saarland.de/Publications/documents/Stark_2020_Mechanising.pdf) for a detailed overview on the different syntactic constructions.

This file system contains a number of example signatures.

#### Compatibility

At the moment, Autosubst 2 comes with the following restrictions:

- Variadic syntax works only with scoped syntax.
- Modular syntax works only with unscoped syntax.
